About

A list of known crowdfunding campaigns where the result would be software released under a free software license. Hopefully this can be a useful resource for free culture academics and would-be free software developers.

This is not a list of open source software for creating a Kickstarter clone. For that try this list.

Some other sites that may be of interest:

  • Bountysource
  • Kickstarter 'open software’ tag (NB. seems to me this is a rather loose designation)

Source/Contributing

The source for this site is available at Github ; patches/pull requests welcome. It is built using Nikola, a static site generator written in Python. If you don’t feel comfortable contributing that way, feel free to just use the contact form.
